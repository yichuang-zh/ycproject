保存选中范围/载入选中范围

此功能无法通过菜单打开，需要使用快捷键打开。默认情况下，inkscape没有为此功能分配快捷键，您需要自行分配快捷键。

此功能有三个命令，分别是：

  1. 保存当前选中范围

  2. 载入保存的选中范围

  3. 删除保存的选中范围

1. 保存当前选中范围 Set selection backup

执行此命令后，当前已经选中的对象(包括节点)会被保存起来，等待以后再次使用。对于需要重复实现的复杂的选择操作，这个功能很实用。

它有点类似剪贴板的功能，或者类似gimp中的选区保存到通道或者选区保存到路径功能，目的是在后面可以重新调取相同的选中范围。

当前文档中只能保存一个选中范围；如果您多次执行此命令，则只有最后一个选中范围会被保存。

2. 载入保存的选中范围 Restore selection backup

执行此命令后，会把之前保存的选中范围调取出来，即把当前选中对象切换为之前保存的选中范围，对于复杂选择操作来说此命令相当迅速。

3. 删除保存的选中范围 Empty selection backup

删掉之前保存的选中范围，之后再执行载入保存的选中范围命令会导致不选中任何对象。

4. 三个命令的快捷键如何设置

默认没有分配快捷键，你需要自行指定快捷键：

菜单【编辑⇨界面⇨快捷键】，名称列找到Select并点击展开，向下滚动可以找到这三个命令。找到命令后，点击快捷键列对应的位置，会显示新建快捷键…，就可以设置快捷键了。

三个命令都设置好快捷键以后，就可以在窗口中通过快捷键来操作复杂的选择操作。
