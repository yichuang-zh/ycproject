⇨克隆

(如果想彻底理解平铺克隆的功能和使用方法，您可能需要有一定的数学基础才行。)

平铺克隆中，缩放中有一个参数为基数Base。此参数的计算方法为：

0: 10^((1+0*0.02)^1-1)=1
19:10^((1+19*0.02)^1-1)=2.4
(1+num_column * 0.02)

基数^((1+列序号*缩放值)^指数-1)=新对象相对于原始对象的尺寸比率

  1. 创建克隆 Create Clone

  2. 创建平铺克隆 Create Tiled Clones

  3. 断开克隆 Unlink Clone

  4. 递归断开克隆 Unlink Clones Recursively

  5. 重新连接副本 Relink to Copied

  6. 选择原始对象 Select Original

  7. 克隆原始路径(LPE) Clone Original Path (LPE)
