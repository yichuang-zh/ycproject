圆形工具

快捷键：E 或者 F5

圆形工具( )可以用来画一个椭圆形、正圆形、饼形、弧形。

操作方法

  1. 画一个椭圆形

用鼠标左键点击圆形工具( )，此时圆形工具被激活，然后鼠标左键在画布上点击并斜向拖动一段距离，然后松开鼠标即可。

  1. 画一个正圆形

与画椭圆形操作类似，只不过在画的时候要按住Ctrl键。

  1. 画一个饼形

  1. 先画一个圆形，然后用圆形工具( )选中该圆形，如下面的左图。

  2. 用鼠标点击并向上拖动红色的小圆点；拖动时保持鼠标在圆形区域外侧移动，不要进入圆形区域内部。

  3. 拖动红色小圆点的位置可以调整饼形的形状；拖动时保持鼠标在圆形区域外侧移动，不要进入圆形区域内部。

  1. 画一个弧形

与画饼形操作类似，只不过要保持鼠标在圆形区域内部移动，不要移动到圆形区域外部。

弧形好像不太像，我们可以这样处理一下：

  1. 选中此弧形，然后按住Shift键不松手，点击窗口左下角的黑色调色板，添加一个黑色描边，并把描边宽度设置为1.00(右键点击窗口左下角描边字样右侧的数字，然后选择1)，如下方左图。

  2. 然后鼠标点击调色板最左侧的叉，取消填充，效果如下方右图。

  1. 然后用圆形工具( )点击此弧形，会显示手柄，如下面左图所示。其中有两个小圆形手柄，用鼠标移动其中一个，可以调整弧形的范围；鼠标要保持在弧形区域范围之内移动，不要移到弧形区域之外。

驿窗注：

在对弧形或饼形进行范围调整时，鼠标的位置很关键：

  1. 如果鼠标移动的最后位置在弧形或饼形的范围之 内，那么调整的结果就是 弧形；

  2. 如果鼠标移动的最后位置在弧形或饼形的范围之 外，那么调整的结果就是 饼形；

  1. 按空格键可以临时切换到选择工具( )，再按空格键则切换回圆形工具。

工具选项

  1. ( Rx )：水平方向圆角半径：对于椭圆形，Rx≠Ry；对于正圆形，Rx=Ry。

  2. ( Ry )：垂直方向圆角半径：对于椭圆形，Rx≠Ry；对于正圆形，Rx=Ry。

  3. ( 起点 )：水平方向到弧形起点的角度。

  4. ( 终点 )：水平方向到弧形终点的角度。

    • 这两个值可以用来精确控制弧形或饼形的范围。

    • 起点值=终点值=0时，弧形变为完整的(椭)圆形。

  1. ( )：用圆形工具( )选中一个弧形后，点击此图标，可以马上转换为饼形。

  2. ( )：用圆形工具( )选中一个饼形或者封闭弧形后，点击此按钮，可以马上转换为开放弧形。

  3. ( )：用圆形工具( )选中一个饼形或者开放弧形后，点击此按钮，可以马上转换为封闭弧形。(封闭弧形也可以通过移动两端的小圆形手柄来调整范围)

  4. ( )：用圆形工具( )选中一个饼形或者开放弧形或者封闭弧形后，点击此按钮，可以马上转换为完整的(椭)圆形

手柄

圆形有4个手柄：其中两个手柄可以对圆形进行缩放操作，另外两个手柄可以调整弧的起点和终点位置。
圆形的手柄如下图所示：
说明如下：

  1. 手柄有两种形状，一种是小正方形，另一种是小圆形。

  2. 正方形手柄分别用来调整长轴方向和短轴方向的尺寸；圆形手柄用来调整弧的起点和终点。调整时使用圆形工具( ) 。

  3. 上面图1中只显示了一个圆形手柄，其实是两个圆形手柄重叠了，用圆形工具( )点击并拖动这个圆形手柄，就可以让重叠的两个手柄位置分开。

  4. 上面图3中上面的方形手柄显示在矩形虚线框之外，这是因为椭圆形的短轴长度超出了弧形的覆盖区域，而方形手柄的位置始终处于轴端位置，不会随弧形覆盖区域变化。

  5. 上面图4中两个方形手柄都显示在矩形虚线框之外，道理同上。
