测量工具

快捷键：M

测量工具( )用来标示测量尺寸。

操作方法

  1. 切换到测量工具( )；

  2. 在某个对象上点击并拖动鼠标。

工具选项

  1. ( 字体大小 )：测量工具显示的数值所使用的字体尺寸。

  2. ( 精确度 )：显示的数值精确到小数后面多少位。

  3. ( 缩放 )：按数值框中的数值比例对显示的数值进行缩放调整。

  4. ( 单位 )：显示的测量单位(自动换算)。

  5. ( )：激活后，只对选中对象进行测量操作。

  6. ( )：测量时，测量的初始点和结束点与最近的对象可能有一定距离。激活此选项，可以忽略这个距离，即不测量这个初始距离和结束距离。

  7. ( )：激活此选项后，会测量对象彼此之间的距离；反之，只测量初始点与结束点距离。

  8. ( )：激光此选项后会显示被隐藏的对象交叉点。对象有部分重合时，如果不激活此工具，会只以最上方对象的边缘为测量对象，正文对象如果边缘被遮挡住，则不进行测量；激活此工具则对被遮挡对象边缘也进行测量。

  9. ( )：激活此选项会测量所有图层，反之仅测量当前图层。

  10. ( )：待更新

  11. ( )：待更新

  12. ( )：点击后，对所有测量点生成参考线。如果不需要参考线，可以点击菜单编辑>删除全部参考线。

  13. ( )：点击后，会把测量结果转换成可编辑的对象；字体保未转曲线。

  14. ( )：待更新

  15. ( 偏移 )：待更新

关于测量工具，inkscape提供了一个扩展，可以用来测量/显示一段路径的实际长度，请先选中一条路径，然后点击菜单 扩展→路径可视化→测量路径，勾选实时预览可以查看预览效果，点击应用按钮即可把长度显示在路径旁边。此长度数字和单位可以选中后删除。
