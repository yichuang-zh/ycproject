缩放工具

快捷键：Z 或者 F3

缩放工具( )用来缩放视图。

操作方法

  1. 切换到缩放工具( )；

  2. 在需要放大查看的位置点击鼠标左键；

  3. 或者，在需要缩放小，按shift不松手，点击鼠标左键。

工具选项

  1. ( )：放大。

  2. ( )：缩小。

  3. ( )：1:1。

  4. ( )：1:2。

  5. ( )：2:1。

  6. ( )：选中对象适合窗口。

  7. ( )：整个绘图适合窗口。

  8. ( )：页面高度适合窗口。

  9. ( )：页面宽度适合窗口。

  10. ( )：页面中心与窗口中心重合。

  11. ( )：上次视图(来自缩放历史记录)。

  12. ( )：下次视图(来自缩放历史记录)。
