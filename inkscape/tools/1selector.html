选择工具

快捷键：S 或者 F1

选择工具( )可以用来选中一个或多个对象，以对其进行各种操作，例如移动、缩放、变形、旋转等。

操作方法

  1. 选中单个对象

用鼠标左键点击选择工具( )，此时选择工具被激活，然后鼠标左键点击需要选择的对象即可选中该对象。

  1. 选中多个对象

用鼠标左键点击选择工具( )，此时选择工具被激活，然后鼠标左键画一个矩形框，需要被选择的对象应该完全被包含在这个矩形框所覆盖的范围之内，然后松开鼠标左键。这样，需要选择的对象就会被选中。

  1. 选中多个对象(shift)

用鼠标左键点击选择工具( )，此时选择工具被激活，然后按住shift键不松手，再用鼠标左键依次点击或框选所有需要选择的对象即可；全部对象选择完成后再松开shift键。

提示

按下shift键不松手再鼠标左键点击某个已选中对象，会把该对象从选中状态变为不选中状态。

  1. 移动对象

选中某个或多个对象后，鼠标左键点击其中一个对象不松手，然后移动鼠标，到达需要的位置后松开鼠标即可。

  1. 缩放/旋转/变形对象

选中某个或多个对象后，对象周围会显示8个控制手柄；在对象上再点击一次鼠标左键，控制手柄形态会发生变化，并且在对象的中心会多出一个中心控制手柄。两种情况如下图所示：

8个控制手柄：第一形态 (左 )；第二形态 (右 )

  1. 左图中，鼠标点击任意一个手柄并拖动，就可以缩放对象。

  2. 右图中，鼠标点击矩形四个角上的任意一个手柄并拖动，就可以旋转对象。

  3. 右图中，鼠标点击矩形四条边上的任意一个手柄并拖动，就可以变形对象。

  4. 右图中，矩形中心的空心十字是旋转和变形的中心点，可以用鼠标移动位置：鼠标左键点击十字并拖动，到达合适的位置后松开鼠标即可。

  5. 以上操作的同时如果按下shift键或ctrl键或alt键时会有更多功能。

工具选项

  1. ( )：全选；当前图层的所有对象都会被选中，等同于快捷键Ctrl+A；当前图层以外的其它图层中的对象不会被选中 。

  2. ( )：所有图层中的对象都会被选中，但被锁定的图层和被隐藏的图层除外。

  3. ( )：取消已经选中的对象，等同于快捷键Esc 。

  4. ( )：选择框功能切换。此图标未激活时，用鼠标框选对象时，需要把对象的所有部分全部包括在选择框内才可以选中；如果某对象有任何一点在框外，则无法选中该对象。此图标激活以后，不需要把对象的所有部分都包括在框内，即只要框线碰到了对象即可选中该对象。

  5. ( )：逆时针旋转90°，等同于快捷键Ctrl+[ 。

  6. ( )：顺时针旋转90°，等同于快捷键Ctrl+] 。

  7. ( )：水平翻转，等同于快捷键H 。

  8. ( )：垂直翻转，等同于快捷键V 。

  9. ( )：堆叠次序(z方向)向上提升到最上面，等同于快捷键Home 。

  10. ( )：堆叠次序向上提升一级，等同于快捷键PageUp 。

  11. ( )：堆叠次序向下降低一级，等同于快捷键PageDown 。

  12. ( )：堆叠次序向下降低到最下面，等同于快捷键End 。

  13. ( X: )：选中对象的左上角的横坐标值(以矩形计算)。

  14. ( Y: )：选中对象的左上角的纵坐标值(以矩形计算)。

驿窗注：Inkscape坐标系的特别之处

Inkscape的坐标系与传统数学意义上的坐标系不完全相同，差别在于纵坐标(Y)：

  1. 传统的数学坐标系中，原点(x=0, y=0)位于坐标系的左下角：

x增加时，点在坐标系中向右移动；
y增加时，点在坐标系中向上移动；

数学坐标系

  1. inkscape坐标系中，原点(x=0, y=0)位于坐标系的左上角：

x增加时，点在坐标系中向右移动；
y增加时，点在坐标系中向下移动；

inkscape坐标系

实际上，大多数计算机图形软件都是这样定义纵坐标Y的，比如photoshop和GIMP。

  1. ( 宽: )：选中对象的宽度值(以矩形计算)。

  2. ( 高: )：选中对象的高度值(以矩形计算)。

  3. ( )：未锁定宽高比。对象在缩放时，宽高比例可以任意调整，不受限制。

  4. ( )：锁定宽高比。对象在缩放时，宽高比例被锁定为固定值，不可调整。

  5. ( )：激活此图标后，对象在缩放时，描边宽度会同时缩放。如果未激活此图标，则对象在缩放时，描边宽度保持不变。

  6. ( )：激活此图标后，对象的圆角在缩放时，圆角半径会同时缩放。如果未激活此图标，则对象在缩放时，圆角半径保持不变。例如缩放圆角矩形。

  7. ( )：激活此图标后，对象在缩放时，渐变会同时缩放。如果未激活此图标，则对象在缩放时，渐变保持不变。

  8. ( )：激活此图标后，对象在缩放时，图案会同时缩放。如果未激活此图标，则对象在缩放时，图案保持不变。
