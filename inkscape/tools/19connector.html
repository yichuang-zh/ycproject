连接工具

快捷键：O 或者 Ctrl+F2

连接工具( )可以用来绘制流程图。使用连接工具绘制的流程图，当被连接的对象移动位置时，连接线会保持连接不断开，且可以将对象标记为避开连接线，这样连接线会自动绕开被标记的对象。

操作方法

  1. 切换到连接工具( )；

  2. 移动鼠标到某个要连接的对象上方，对象中心会显示一个小方块；

  3. 点击小方块，并拖动鼠标到下一个要连接的对象上；

  4. 下一个要连接的对象也会显示一个小方块，点击此小方块；

  5. 两个对象连接完成。

  6. (小方块为连接线的控制手柄)

  7. (用连接器工具点击连接线会显示连接线的控制手柄)

  8. (用连接器工具点击控制手柄，可以断开或重新连接对象)

  9. (可以通过工具选项中的 间隙 数值来设置连线与对象的最小距离)

  10. (可以通过工具选项中的( )和( )选项来设置连接线是否绕开对象)

  11. (可以通过工具选项中的 弯曲 数值来设置连接线拐角的圆角程度)

  12. (可以通过菜单 对象→填充和描边 中的描边样式→标记来为连接线设置箭头)

工具选项

  1. ( )：连接线绕开选中的对象。

  2. ( )：连接线不绕开选中的对象。

  3. ( )：连接线为直角折线。

  4. ( 弯曲 )：连接线直角拐角的圆角程度。

  5. ( 空隙 )：连接线与连接对象之间的最小间距。

  6. ( )：自动调整连接的网络。

  7. ( 长度 )：自动重新排列时，此选项可以设置连接线的近似长度。

  8. ( )：待更新

  9. ( )：禁止重叠。

  10. ( 以下为 对齐与分布 中自动排列相关的选项 )

    图 19-1: 对齐与分布 中选项与图标

  11. ( )：自动调整连接的网络。

  12. ( )：交换选取对象的位置：选取位置。

  13. ( )：交换选取对象的顺序：堆叠顺序。

  14. ( )：交换选取对象的顺序：顺时针旋转。

  15. ( )：两个方向随机移动对象。

  16. ( )：解散对象，以边边距离相等为目标。

  17. ( H )：对象边缘间最小水平间距。单位为像素。

  18. ( V )：对象边缘间最小垂直间距。单位为像素。

  19. ( )：点击此图标，会按H值和V值来移动选中对象，调整对象间距；这个功能可以快速分离重叠的对象。如果对象间的距离已经大于H值/V值，则不移动对象。
