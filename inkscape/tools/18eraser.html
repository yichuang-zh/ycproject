橡皮工具

快捷键：Shift+E

橡皮工具( )用来擦除图形对象的一部分，或者擦除对象整体。

操作方法

  1. 取消选中任何对象；

  2. 切换到橡皮工具( )；

  3. 在对象上点击并按住鼠标左键不松手同时移动鼠标指针的位置。

工具选项

  1. ( )：对象整体擦除。删除此工具碰到的对象的整体。忽略群组，即群组中只删除碰到的对象，群组中未碰到的对象不会被删除。

  2. ( )：局部擦除。从路径或图形上删除鼠标经过的部分，不是整体删除；未碰到的其它部分会保留。默认情况下，对象即使被断开成多个部分，所有部分仍是一个整体，比如移动时所有部分会整体移动；除非激活分解对象图标( )。

  3. ( )：剪切擦除。从对象上切下鼠标经过的部分。此选项不会补充描边，而上面的局部擦除选项( )在擦除后，对象如果原本有描边，擦除后会把描边补上。

  4. ( 宽度 )：鼠标擦除时的宽度，范围0-100。数值是相对值，相对屏幕可见画布尺寸，不是绝对值。

  5. ( )：如果使用绘图板，那么可以激活此选项后，触笔的压力变化会导致擦除轨迹的宽度发生相应变化。此选项对鼠标操作无效。

  6. ( 细化 )：范围-100-100。轨迹移动速度与粗细变化的关系。0表示无关；大于0时，轨迹移动速度越快，宽度越细；小于0时，速度越快，宽度越粗。此选项对鼠标及绘图板都有效。

  7. ( 端点 )：端点的外端形状，范围0.00-5.00。0.00表示直线；1.00表示圆形；5.00时，是圆形且突出，近似尖圆(或近似水滴形)。

  8. ( 抖动 )：模拟笔尖轨迹抖动效果，范围0-100。数值越大，轨迹抖动越厉害。

  9. ( 质量 )：轨迹落后于鼠标指针的程度，范围0-100。数值越大，轨迹落后得越远。

  10. ( )：分解对象。此选项在使用局部擦除选项( )时可以使用。激活后，被局部擦除的对象如果断开，各个部分会分别独立，不再是整体。
