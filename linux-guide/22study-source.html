学习Linux可用的免费资源列表

自学Linux系统，包括自学开源应用，推荐使用以下资源：

  1. 论坛

  2. QQ群/聊天群/聊天室

  3. 视频

  4. wiki/文档

1.论坛

论坛是最经典的一种学习渠道，不仅可以提问，还可以搜索以前已经问过的问题。而且，论坛通常都会有一些知识丰富的人回答问题，非常有效。(通常需要邮箱注册)

国内几个Linux/开源相关论坛如下(不完全整理)：

  1. Arch Linux中文论坛

  2. Debian中文论坛

  3. Deepin官方论坛

  4. LibreOffice中文论坛

  5. openSUSE中文论坛

  6. Ubuntu中文论坛

其中Ubuntu中文论坛的用户数量最大，活跃人数也最多。

上面仅列举了部分中文论坛，如果英文好的话，可以去试试英文论坛。

2. QQ群/聊天群/聊天室

各种即时聊天工具也非常方便，比如QQ群。与论坛相比有一个不好的地方是无法有效检索过往的信息。

如果想加入某一个聊天群，可以直接打开QQ搜索群名称，比如搜索Debian，就会列出Debian群，选择加入即可。

3. 视频

各种视频网站上会有他人分享的视频，也很方便。与文字资料相比，视频更形象，但不像文字那样可以快速阅读，快进和倒退感觉不如文字方便。

目前国内比较常用的视频学习网站是B站，这里可以搜索到很多学习视频。

4. wiki/文档

wiki最著名的就是Arch wiki，这里有非常丰富的Linux知识，未接触过Linux的人刚开始可能不习惯这里的文档，不过一旦习惯之后就会发现，这里是一座Linux知识宝库。另外，其它发行版也都有wiki可以查看。

文档，一方面是发行版官方提供的文档，比如Debian文档，这里有比较全面的Debian系统知识。

另一方面，可以参考其他人总结整理后分享出来的学习文档，有关于Linux操作系统方面的文档，也有关于开源应用方面的文档。尤其那些相对系统性的文档，知识点也相对要全面一些。

在提问题时，注意一下提问的方法。不了解的话，可以参考提问的智慧。
