一. 为当前用户添加sudo权限

默认情况下，好像当前普通用户并没有sudo权限(管理员权限)，所以可能无法修改一些系统级的设置，包括/etc/apt/sources.list这个软件源。

如果不清楚是否有sudo权限，可以在命令行终端中运行id命令检查一下：

id

输入id并回车后，如果输出的内容有sudo字样，就表示已经有sudo权限了。如果没有sudo字样，请按以下方法为当前用户添加管理员权限：

在命令行运行下面的命令：

su -
adduser peter sudo

上面的第一条命令需要输入超级管理员密码(root密码)。两条命令运行之后，当前用户peter就有了管理员权限。

如果你当前用户名不是peter，记得把peter换成你的用户名。

二. 软件源

(推荐使用中国科学技术大学、清华大学等国内软件源)

Debian系统默认使用的软件源是Debian官方源，服务器不在国内，所以安装软件、升级系统时可能会比较慢。而国内软件源最大的好处就是速度快，这也是系统安装好以后先要修改软件源的主要原因。

软件源主要就是指服务器地址，Debian把需要使用的服务器地址放到了一个文件中：

/etc/apt/sources.list

只需要修改这个sources.list文件的内容即可。

三. cpu补丁(微码)

Debian12已经可以自动安装，不需要手动处理。Debian11或更早的版本，需要手动安装。

cpu补丁其实就是一个软件包，不同的cpu，软件包名称也不相同。

如果是Intel的CPU，请在命令行终端中执行下面的命令：

sudo apt install intel-microcode

如果是AMD的CPU，请在命令行终端中执行下面的命令：

sudo apt install amd64-microcode

具体安装哪个取决于电脑使用的是哪种cpu，安装完成后重启系统即可。

五. 防火墙

防火墙需要安装两个软件包：ufw和gufw。其中，gufw仅仅是ufw的图形界面。在命令行终端中执行下面的命令安装：

sudo apt install ufw gufw

安装好以后，在开始菜单中可以找到防火墙，启动后简单设置即可。

防火墙和杀毒软件的详细设置说明，请参考系统安全/防火墙/杀毒。
