可停靠对话框→颜色

没有快捷键。

可停靠对话框→颜色

颜色对话框也叫前景色/背景色对话框，其功能与工具箱中的前景色/背景色工具基本相同，本节仅说明不同的部分，相同部分请参考工具箱中的前景色/背景色工具。

提示

此工具不是用来调整图像的颜色，而是用来创建一个颜色；您可以把创建的颜色填充到图像中。

  1. ( ) GIMP调色板：与工具箱中的前景色/背景色工具相比，一是右侧多了六个通道按钮，二是色相条会变成通道值。

  • 在点击六个通道按钮之前，大色块右侧的纵向细 色相条 等同于工具箱中的前景色/背景色工具的色相条，可以选择色相{Hue}；

  • 在点击六个通道按钮之中的RGB中的某一个之后，纵向细 色相条 会变成指定色相的 数值条。即，上下移动滑块时，等同于只调整指定通道的色值，其它通道色值不变；

    例如，通道按钮我们激活R，此时移动 数值条 的滑块，那么当前RGB颜色仅R色值会发生变化，B和G的色值保持不变。

    如果六个通道按钮中激活的是LCh中的L或C或h，那么道理相同，仅激活的通道值会随滑块移动而改变，LCh模式的另外两个通道值不会发生变化。

  1. ( ) 水彩调色板：这个与工具箱中的前景色/背景色工具相同。

  2. ( ) CMYK调色板：这个与工具箱中的前景色/背景色工具相同。

  3. ( ) 轮：这个与工具箱中的前景色/背景色工具相同。

  4. ( ) 色板：这个与工具箱中的前景色/背景色工具相同。

  5. ( )缩放：这里可以详细显示所选颜色的通道值；也可以通过修改这里的通道值来指定当前颜色。

  6. (其它内容与工具箱中的前景色/背景色工具相同)
