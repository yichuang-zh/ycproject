可停靠对话框⇨指针

没有快捷键。

指针信息对话框( )可以实时显示鼠标指针的坐标位置，以及所选颜色模型中指定像素的通道值。

概述

  1. 像素：显示指针所在位置的像素的坐标，以距离原点(画布左上角)的像素距离为单位。

  2. 单位：显示距离原点的距离，单位是英寸。

  3. 选区：X和Y是选区矩形框左上角的坐标。宽和高是这个选区矩形框的宽度和高度。(如果选区是不规则选区，那么这里的信息基本没啥用)

  4. 通道值：根据指定的颜色模型，显示相应的通道信息。两个下拉菜单内容相同，这样可以方便您使用不同的颜色模型对同一个像素的色值进行对比。

    • 像素：像素的通道值。如果图像是RGB模式，则等同于下面的RGB(0..255)；如果图像是灰度模式，会显示灰度值和Alpha值；如果图像是索引模式，则等同于下面的RGB(0..255)，但Hex会换成索引。

    • RGB%：RGB色值(通道值)，A表示Alpha值；以百分比方式显示。Hex表示像素颜色的十六进制值，主要用于HTML/CSS。

    • RGB(0..255)：像素的Red、Green、Blue、Alpha、Hex值，其中RGBA值的范围为0到255，Hex表示像素颜色的十六进制值。

    • HSV：以度数显示像素的色相Hue，以百分比显示像素的饱和度Saturation、明度 Value 和Alpha。(可以参考色相环的HSV表示方法)

    • CMYK：CMYK模式，以百分比的形式显示像素的C、M、Y、K和Alpha值。

    • CIE LCh：CIE LCh模式，显示像素的L、C、h和Alpha值。( 高级用户 )

    • CIE LAB：CIE LAB模式，显示像素的L、a、b和Alpha值。( 高级用户 )

    • CIE xyY：CIE xyY模式，显示像素的x、y、Y和Alpha值。( 高级用户 )

    • CIE Yuꞌvꞌ：CIE Yuꞌvꞌ模式，显示像素的Y、uꞌ、vꞌ和Alpha值。( 高级用户 )

  5. 位样合并：勾选此选项后，会对所有可见图层进行采样；如果不勾选，仅对当前活动图层进行采样。
