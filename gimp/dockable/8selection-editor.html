可停靠对话框⇨选区编辑器

没有快捷键。

选区编辑器对话框( )可以更直观地显示当前活动选区，并对选区进行管理。

选区编辑器对话框中，白色为选中区域，黑色为未选中区域，灰色为部分选中区域。

选区编辑器对话框中的按钮

  1. ( )选择全部：创建一个选区，包括当前图层所有内容，等同于Ctrl+A。

  2. ( )摒弃选区：取消当前选区，等同于Ctrl+Shift+A。(浮动选区不受影响)

  3. ( )反转选区：反转当前图层的选区，即选择与之前完全相反的区域，等同于Ctrl+I。

  4. ( )选区到通道：把当前选区保存到通道，等同于菜单【选择⇨保存到通道】。

  5. ( )选区到路径：把当前选区转换为路径，选区不变，等同于菜单【选择⇨到路径】。(点击时按Shift键会显示高级选项)

  6. ( )勾画选区：勾画选区轮廓，等同于菜单【编辑⇨勾画选区】。点击时按Shift键会直接按上一次的参数直接勾画。
