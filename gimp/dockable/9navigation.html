可停靠对话框→导览

没有快捷键。

可停靠对话框→导览

导览对话框( )可以让您更方便地查看图像的某个区域或整体。(视图→导览窗口)

概述

  1. 截图中白色矩形表示当前图像窗口显示的内容，白色矩形区域以外的内容在图像窗口中因放大而未显示。

  2. 白色矩形可以拖动，图像窗口显示的内容会同步变化。

  3. 点击白色矩形以外的区域，白色矩形会立刻切换到点击的位置，图像窗口显示的内容会同步切换。

  4. 按Shift键同时滚动鼠标滚轮，白色矩形会水平移动，图像窗口显示的内容会同步变化。

  5. 按Alt键同时滚动鼠标滚轮，白色矩形会垂直移动，图像窗口显示的内容会同步变化。

  6. 窗口底部有一个滑块和六个按钮，可以调整图像显示尺寸或图像窗口尺寸。

导览对话框中的按钮

  1. 缩放滑块可以直接用鼠标拖动。

  2. ( )缩小：缩小显示，等同于 - 。

  3. ( )放大：放大显示，等同于 Shift + +。

  4. ( )1:1：如果未勾选菜单 视图→点对点，以打印尺寸显示；如果已勾选菜单 视图→点对点，则图像窗口中图像的每个像素会与屏幕像素(硬件)一一对应。

  5. ( )调整缩放比率以使图像完整显示：调整缩放比例，使当前图像窗口可以完全显示整个图像，图像窗口尺寸不变。

  6. ( )调整缩放比率以使窗口最佳化使用：调整图像缩放比例，使图像宽与高有一个方向完整显示，图像窗口尺寸不变。

  7. ( )缩小图像窗口到图像显示的尺寸：调整图像窗口尺寸，允许图像完全显示且图像的缩放比例不变。(也有可能放大图像窗口)
