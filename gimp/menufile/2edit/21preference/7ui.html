语言

GIMP能够自动检测操作系统的语言设置。您可以修改GIMP的界面语言，比如从简体中文界面改成英文界面，或者从英文界面改成简体中文界面。

驿窗注：双语 | 中英对照 | 双语GIMP同时运行 | GIMP汉化

通常我们切换GIMP的界面语言时，是进入菜单【首选项⇨界面⇨语言】，把语言修改后重新启动GIMP。这样的操作方法没有问题，但对于需要同时显示两种语言界面的人来说(比如翻译人员)，非常不方便。

GIMP在Linux环境中支持多实例运行(命令参数 -n)，所以，您可以利用这一点，来同时运行两个语言界面的GIMP，这样就避免了频繁重复关闭和启动GIMP的麻烦。

根据GIMP安装方式的不同，下面我们讨论情况：

  1. 官方仓库 (sudo apt install gimp)

  2. appimage

  3. flatpak方法如下：

  1. 官方仓库版本的GIMP sudo apt install gimp：

    这种情况下，您打开终端，输入 gimp 后回车，就会启动GIMP。

第一步，创建文本文件： 请用文本编辑工具(比如gedit就可以)创建两个文本文件，文件名分别为gimpcn和gimpen，内容如下：

  • 文件 gimpcn：

#gimpcn, Chinese UI
#!/bin/sh
LANGUAGE=zh_CN gimp -n

  • 文件 gimpen：

#gimpen, English UI
#!/bin/sh
LANGUAGE=en gimp -n

这两个文件请保存在你的home目录中，接下来我们用这两个文件来启动GIMP。

第二步，打开终端，切换到你的home目录(即gimpcn文件和gimpen文件所在的目录)，输入 sh gimpcn，回车，就可以启动简体中文界面的GIMP。此时再打开一个新的终端，目录同样是你的home目录，输入 sh gimpen，回车，就可以启动英文界面的GIMP。

好了，现在您已经同时运行了简体中文界面GIMP和英文界面GIMP。如果需要，您还可以按照上面的方法再同时运行繁体中文GIMP、韩文GIMP……

  1. appimage版本的GIMP：

    方法同上，也是创建两个文本文件，只不过文件内容稍做修改。gimpcn文件和gimpen文件的内容分别如下：

    • 文件 gimpcn：

    #gimpcn, Chinese UI
#!/bin/sh
LANGUAGE=zh_CN /home/ycp/tmpfs/GIMP_2.10.38.AppImage -n

    • 文件 gimpen：

    #gimpen, English UI
#!/bin/sh
LANGUAGE=en /home/ycp/tmpfs/GIMP_2.10.38.AppImage -n
    其中：
    /home/ycp/tmpfs/ 是appimage文件所在的目录；
    GIMP_2.10.38.AppImage 是appimage文件的文件名。

  2. flatpak版本的GIMP：

    faltpak版本的GIMP不用创建文本文件，直接在终端运行就可以，方法如下：

    • 打开第一个终端，启动GIMP：flatpak run org.gimp.GIMP

    • 打开第二个终端，启动GIMP：flatpak run org.gimp.GIMP -n

      两个GIMP的语言是相同的，现在把第二个终端运行的GIMP的语言改成需要的语言，假设第一个终端启动的GIMP是简体中文，那么这里第二个终端启动的GIMP可以在首选项中改成英语，然后退出这个GIMP，然后在这个终端中重新运行第二条命令。注意，只是退出GIMP，不要关闭运行GIMP的第二个终端。

    • 第二个终端再一次启动GIMP：flatpak run org.gimp.GIMP -n

      这次启动的GIMP已经是新设定的英语了，与第一个终端启动的GIMP可以双语对照使用。

使用紧凑滑块

这个是指GIMP中所有滑块条的高度，紧凑滑块会更节约屏幕空间。例如，如果您已经打开了矩形选择工具的工具选项的羽化边缘-半径滑块，那么，勾选或取消勾选此选项可以实时看到滑块高度的变化。

预览

默认GIMP会显示图层和通道的预览(缩略图)。如果取消勾选启用图层和通道预览和启用图层组预览这两个选项，那么GIMP就不会显示缩略图。建议一直保持勾选状态。

另外，下面的默认图层和通道预览大小、撤销预览大小、导航预览大小选项可以设置缩略图的尺寸。

快捷键

按Alt+字母可以激活最顶层的菜单，比如按Alt+F可以激活文件菜单，Alt+C可以激活颜色菜单

使用动态快捷键

GIMP可以实时创建键盘快捷键：让指针悬停在所需的菜单项上时按下按键即可。默认情况下，此功能被禁用，因为它可能会导致用户意外地改变现有快捷键。如果想启用此功能，请勾选此项。

驿窗注：

动态快捷键不能修改顶级菜单，只能修改顶级菜单下面的菜单项。比如，Alt+F是打开文件菜单的快捷键，而文件菜单是顶级菜单，所以动态快捷键功能无法修改Alt+F。而文件菜单下面的另存为、导出为等菜单项的快捷键则可以修改。

配置快捷键

这个大按钮点击后可以修改快捷键。

退出时保存快捷键

如果勾选此项，您对快捷键所做的修改在退出GIMP时会被保存，以后再打开GIMP时可以继续使用。

立即保存快捷键

马上保存修改的快捷键。

重置快捷键为默认值

把快捷键重置为GIMP默认状态。

删除所有快捷键

把快捷键都删除。请慎重使用此功能，因为它会删除GIMP的全部快捷键！

如果不小心删除了全部快捷键，可以点击重置快捷键为默认值按钮来恢复快捷键。

注意

如果您对快捷键做了自定义设置，那么点击删除所有快捷键按钮之后，将无法恢复这些自定义的快捷键，只能恢复到GIMP默认的快捷键，除非你已经提前备份了存储快捷键定义的文件，通常是个人目录下的 menu.rc 文件。

menu.rc 文件的存储位置，请参考【首选项➡键盘快捷键文件】
